# TaskManager Web

Mini app en HTML/CSS/JavaScript para gestionar una lista de tareas.

## Funcionalidades (por versiones)
- v1.0.0: añadir tareas, marcar como completadas, eliminar tareas
- v1.0.1: hotfix de borrado (simulación)

## GitFlow (resumen)
- main: producción (solo versiones estables con tags)
- develop: integración de desarrollo
- feature/*: nuevas funcionalidades
- release/*: preparación de versión
- hotfix/*: corrección urgente desde producción
